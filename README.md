
# Indexed Resource Repository

The files in /data/ are sourced from
[music pack at archive.org](https://archive.org/details/keygen-music-2020-03-pack) and
[modarchive.org](https://modarchive.org/index.php?faq-licensing), in unmodified form.

Both data and index are intended to be used by the web player.

**Open and play it: [pl4yg3n.github.io](https://pl4yg3n.github.io/)**
